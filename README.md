## 基于PHP医院预约挂号系统小程序+LW(源码+数据库)

## 一、系统介绍
根据医院预约挂号系统小程序的功能需求，进行系统设计。
前台功能：用户进入系统可以实现首页、医生信息、医院资讯、我的等内容，在我的页面可以对个人中心，预约挂号等功能进行详细操作；
后台主要是管理员和医生，主要包括首页、个人中心、用户管理、医生管理、科室分类管理、医生信息管理、预约挂号管理、系统管理等功能；
